### [Практика 1: CSS-препроцессоры (SASS)](Practice1/)
Карточка товара с использованием переменных, миксинов и вложенности селекторов.

### [Практика 2: Сервер на Node.js + Express](Practice2/)
API и CRUD.

### [Практика 3: Тестирование API в Postman](Practice3/)
Тестирование созданного и выбранного (погода) API с помощью Postman. 

### [Практика 4: API + React ](Practice4/)
Создание полноценного веб-приложения: бэкенд на Express (CRUD для товаров) и фронтенд на React. Реализация интернет-магазина с возможностью просмотра, добавления, редактирования и удаления товаров через API. Стилизация с использованием SCSS.

### [Практика 5: Авто-документация через Swagger](Practice5/)
Доработка бэкенд части веб-приложения: подключение Swagger для авто-генерации документации API. Все CRUD операции и интерфейс на http://localhost:3000/api-docs.

### [Практика 7: Базовые методы аутентификации](Practice7/)
Реализация серверного приложения на Node.js с регистрацией и входом пользователей. Хеширование паролей с использованием bcrypt. Создание маршрутов для управления товарами с разделением прав доступа.

### [Практика 8: JWT авторизация](Practice8-10/)
Доработка серверного приложения из практики 5 - добавление JWT-токенов для авторизации. Реализация защищенных маршрутов для работы с товарами (создание, редактирование, удаление доступны только авторизованным пользователям). Фронтенд: окна входа и регистрации.
